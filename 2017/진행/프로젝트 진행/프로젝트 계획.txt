- 웹
    - 학급운영 및 학교생활지원 시스템 구축
        - 주요 기능
            - 일정 관리
                - 명칭, 기한 입력받아 달력 / 목록으로 표시
                    - 받는 정보는
                        1. 일정 이름
                        2. 일정 기한
                        3. 일정 설명
                - 반 별로 공통 과제는 각 반 관리자가 입력할 수 있도록 통합, 연계 서비스 제공
                    - 즉, 한 반이 다 같이 해당되는 과제는 관리자가 올리면 반 학생 모두에게 적용
                    - 일부 학생만 해당되는 과제는 각자 올려서 각자 관리
                - 과제 알림 서비스 제공
                    - 남은 시간이 정한 시간보다 적게 남으면 알림
            - 학급비 관리
                - 각 학급의 관리자가 학급비 추가, 수정 및 삭제
                - 현재까지 모은 학급비와 앞으로 모일 학급비 계산해서 표시
                - 받는 정보는
                    1. 학급비를 제출하는 사람(학번)
                    2. 학급비 금액
                    3. 학급비 설명
                    4. 학급비 적용 시작 일자
                    5. 학급비 제출 완료 일자(빈 칸이면 미제출)
            - 레주메
                - 본인의 레주메 관리
                    - 자신의 레주메를 한 눈에 볼 수 있는 기능
                    - 레주메에 정보 추가, 수정, 삭제
                    - 레주메를 엑셀 파일(csv)로 저장할 수 있도록 하는 기능
                    - 레주메 추가 시 받는 정보는
                        1. 정보 분류
                        2. 분류에 따른 내용
                            - 자격증, 시험 같은 경우 취득일자
                            - 봉사, 인턴, 대회, 프로젝트 같은 경우 소개 / 진행 기간
                            - 관심분야, 자기소개, 취미 같은 경우 해당하는 글
                            - 독서 같은 경우 책 이름 / 읽은 기간
                    - 레주메 추가는 허가제로 학급 임원의 허가가 있어야 추가됨
                - 메인 페이지에 레주메 공개
                    - 공개 설정을 하면 메인 페이지에 레주메 보기 버튼이 활성화됨
            - 게시판
                - 자유게시판
                - 공지사항 게시판
            - 게임
                - 코딩
                - 알고리즘 문제풀기
                - 네트워크 문제풀기
                - 리눅스 문제풀기
                - 수학 문제풀기
                - 기타 등등
                    - 모든 게임은 시스템 사용자끼리 매칭
                    - 사용 가능 시간은 쉬는시간, 식사시간, 자습시간이며, 게임 중 시간이 끝나면 무승부처리
                    - 승패에 따라 포인트 차감 및 부여
                    - 포인트는 상점에서 여러가지 아이템을 구입할 때 사용 가능
                        - 현재 생각해 놓은 것은
                            1. 하루 지각비 면제
                            2. 아이스크림, 과자 등 먹을거리
                            3. 필기구
                    - 포인트는 선생님 또는 관리자가 지급할 수 있음
                - 랭킹 시스템
            - 학급 신문고
                - 건의사항
                    - 건의사항을 입력하면 학급 임원의 선택에 따라 폐기 또는 담임 선생님 또는 학생회 또는 기숙사 자치회에 송달



        - 액터
            - 재학생
                - 레주메 관리
                - 자유게시판 활동
                - 개인 일정, 과제 관리
                - 과제에 대한 알림을 받음
                - 게임, 포인트로 상점 이용
            - 학급 임원(관리자)
                - 레주메 추가 신청 판단
                - 학급 신문고의 건의사항 판단
                - 위 정보가 추가되면 알림
               
                - 학급 과제 관리
                - 공지사항 게시판 활동
                - 게임 포인트 관리
            - 학급비 관리자
                - 학급비 관리
            - 담임 선생님
                - 학급 신문고의 건의사항이 학급 임원을 거쳐 옴
                - 위 정보가 추가되면 알림

                - 공지사항 게시판 활동
                - 게임 포인트 관리

            - 기숙사 자치회
                - 기숙사 관리 시스템과 협업
            - 학생회
                - 학생회 GitHub에 이슈 추가하는 방식으로 건의
