-- 동아리 활동 계획 --
동아리는 매주 금요일 7, 8, 9, 10교시에 활동해.
우리는 7, 8교시와 9, 10교시를 나눠서 두 가지를 동시에 학습할 예정이야.

우리 동아리는 Info라는 이름에 걸맞게 정보보안에 대한 도움말과 같은 활동을 할 예정이야. 모든걸 알려줄 순 없지만 최소한 정보보안이 무엇이고, 어떤 길로 공부해야 하는지 구별은 할 수 있게 해 줄 거야.

2학년이 주도한다고 하지만 사실 다들 입문단계라서 믿음직한 면이 좀 떨어질 수도 있어 ~.~
그래도 최대한 열심히 준비해서 최선을 다해 운영하도록 노력할게!

1학기 초반 (3~4월) :
7, 8교시 :
1학년을 위해, 2학년은 복습 겸 C언어를 "처음부터" 소켓 통신까지 공부할 예정이야.
주로 기본 개념 학습, 자주 헷갈리는 것들에 관해 다룰 거야.
이론보단 실습 위주로.
9, 10교시 :
네트워크에 대해 공부할 예정이야.
아무래도 2학년 정보보안과에서 현재 학습중인 내용(CISCO 네트워킹)을 다루게 될 거라고 생각해.
1학년 정보보안과 희망자에겐 정말 좋은 기회가 아닐까?

1학기 중반 (5~6월) :
7, 8교시 :
C언어 학습이 진행되어 소켓 통신을 배우고 나면 윈도우 / 리눅스 환경에서의 소켓 통신을 자세히 다루고 네트워크 해킹 프로그래밍을 C언어로 학습해 볼 거야.
네트워크에 대해 어느정도 학습한 후니까 잘 따라올 수 있을거야.
9, 10교시 :
네트워크 학습을 계속 할 예정인데, 학교 진도에 맞춰서 공부하면 아마 남는 시간이 있을텐데 그때 MySQL을 이용해서 데이터베이스를 공부해 볼 거야.

1학기 후반 (7월~방학 전) :
7, 8교시 :
네트워크 해킹 프로그래밍을 마치면 바로 웹 공부를 시작해 볼 거야.
일단 HTML, CSS, Javascript를 학습할 예정이야.
9, 10교시 :
역시 네트워크 학습을 계속 할 예정이지만, 시간이 남으면 웹서버 관련 언어들(asp, jsp)에 대해 학습할 예정이야. 그리고 8교시와 연동해서 간단한 웹 어플리케이션을 만들어 볼 거야.

여름방학 중
여름방학이 그리 길진 않겠지만 무작정 놀 수는 없겠지? 학교에 나오는 여름방학을 이용해 리버싱에 대해 스스로 공부할 수 있게 될 정도까지는 공부해보고 싶은데 시간이 될진 잘 모르겠다.

2학기 초반(개학 후~9월)
7, 8교시 :
방학 전에 공부하던 웹 프로그래밍 공부를 계속해서 할 예정이야. 어느정도 한 후에는 자신의 웹 애플리케이션을 만들어 볼 예정.
9, 10교시 :
네트워크 공부는 끝이 없을 것 같아.
시간이 남으면 웹 애플리케이션을 만들기 위한 공부를 추가적으로 할 예정.

2학기 중반(10월~11월)
7, 8교시 :
웹 공부가 마무리되면 리눅스 공부를 좀 하게 될 거야.
9, 10교시 :
네트워크..말 안해도 알지?
시간이 남고 리눅스 공부좀 하면 해킹 문제풀이 사이트를 통해 문제를 좀 풀어볼 예정이야.

2학기 후반(12월~종업식 전)
이 기간은 혹시모를 일정 변경을 위한 빈 시간!
계획대로 되면 참 좋지만 그래도 대비는 해야되니까..

모든 학습은 동아리 카페를 개설해 우리의 열정을 일기 형식으로 남기며 진행할 거고, 질문은 언제나 환영하는 활동을 하게 될 거야!

혹시라도 힘들겠다고 생각한다면 너무 걱정하지 않아도 돼.
힘들면 쉬어가면 되고, 무엇보다 우리학교 형누나들이 참 착하거든 ㅋㅋㅋㅋㅋㅋㅋㅋ
